# Prienik kružníc

Modulárny Python projekt, ktorý zistí, či sa dve kružnice pretínajú.
Ak sa pretínajú, program zistí koľko majú spoločných prienikov (0, 1 alebo 2). 
Výsledok vypíše do terminálu a vykreslí do grafu.

Tu je ukážka, keď sa kruhy nepretínajú:

![Malý náhled grafu](circles_obrazky/circles_ziadny_prienik.png)

Aby sme spustili testy, ako prvé sa musíme uistiť, že sa nachádzame v správnej zložke.
Do terminálu napíšeme cd circles_project, následne uv pip install -e".[dev]", a ako posledné uv run pytest.
